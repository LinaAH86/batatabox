Setup Guide
Connect JSONBin and deploy to Vercel
🚀 Get live in 5 minutes
1
Create a free JSONBin account at jsonbin.io → go to API Keys and copy your Master Key
2
Create a new Bin → click Create Bin → paste the starter JSON below → save. Copy the Bin ID from the URL.
3
Open index.html and admin.html → find the CONFIG section at the top of the <script> → paste your JSONBIN_BIN_ID and JSONBIN_API_KEY
4
In admin.html also set ADMIN_PASSWORD to your chosen password
5
Deploy to Vercel — push your folder (index.html + admin.html + logo-1.png) to GitHub, then import in Vercel. Done! 🎉
6
Generate your QR code at qr.io pointing to your Vercel URL — print and place on tables
